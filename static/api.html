API for BfloTrees

This is the HTTPS API to retrieve data from BfloTrees, an app for exploring and annotating the urban canopy of Buffalo, NY.

URL

https://bflotrees.mandelics.com/opendata/trees?

Parameters

  • t: table name: species, allTrees, or notes; default allTrees
  • q: JSON-encoded sequelize query object, e.g. {"where": {"latitude": {"$between": [42.88888,42.88889]}}, "order": [["latitude", "DESC"]]}
Parameters should be urlencoded for GET requests, or sent in the body of POST requests.

Response

JSON-encoded array of results.

Example

GET https://localhost:9001/opendata/trees?q={%22where%22:{%22latitude%22:{%22$between%22:[42.88888,42.88889]}},%22order%22:[[%22longitude%22,%22DESC%22]]}

URL

https://bflotrees.mandelics.com/opendata/tree?

Parameters

  • t: table name: species, allTrees, or notes; default allTrees
  • q: JSON-encoded sequelize query object, e.g. {"where": {"id": 300}}
Parameters should be urlencoded for GET requests, or sent in the body of POST requests.

Response

JSON-encoded result object.

Example

GET https://localhost:9001/opendata/species?t=species&q={%22where%22:{%22id%22:300}}
